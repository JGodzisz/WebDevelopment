What I leanred in CTI 110

HTML

In our HTML section I learned many of the basic skills needed for programing web pages.
I learned how to use tags. How to insert text and images into pages.
We learned about applying emphasis, strong, and
line break tags.

My Favorite part of HTML was creating code in notepad++ and sublime text then,
saving it and checking how it looked when opened in different web browsers.

The hardest part of HTML was learning how to organize my code
so it was not overwhelming to look at. Also finding and using the basic set up of
HTML pages such as DOCTYPE and CHARSET= posed a challenge since it was not
reviewed in the lesson. I also found I needed to work on better habits such as
capitalizing DOCTYPE and not HTML

One example of what I created with HTML is a my hobbies page.
This was part of the final project for both HTML and CSS.

CSS

In our CSS section I learned how to add style to my web pages.
Using style tags, class, and id tags I learned how to customize
a lot of elements of my pages. I learned how CSS works with HTML to create
the overall look of the page.

My favorite part of CSS was researching new ways to style my page. With most
reference guides you can find all kinds of new ways to be creative with each
page element.

The hardest part of CSS was working with alignment and positioning.
I consistently wanted specific elements aligned at specific points
and learning about float properties was more advanced. However, overcoming
the challenge was very rewarding.

One example of what I created with CSS was my blog project. I kept updating
while I learned java and really enjoyed the freedom this project gave me.

Javascript

In our Javascript lessons I learned about functions and "drawing" with code.
I learned how to animate shapes and how to use variables to make repeated
code easier to work with.

My favorite part of Javascript was animation and creating my own functions.
Being able to bring your code to life is very exciting.

The hardest part of learning Javascript was also animation. I struggled with
parts of animation when I created to many variables that where dependent
on other variables.

One example of what I created with Javascript is my shooting star project. I felt
very accomplished with how this project turned out. I had not learned how to
draw shapes yet and used the documentation provided in Kahn Academy to figure
out how to make each shape. The comet that passes became a defining moment in
the class and my career that I will always remember.

SQL

In SQL I learned the Standard Query Language. This is an extremely important language
used with databases that any IT professional needs to know. I learned how to build a
simple database and how to preform and organize a query.

My favorite part of SQL was going outside of Kahn Academy for our projects. Being able to
use another program helped me build confidence that I understood the language.

The hardest part of SQL was remembering what to capitalize and what not to. This caused me
problems a few times during my projects. Also it is hard to retain because the
length of time spent on SQL was quite short.

One example of what I created with SQL was a movie database with my favorite movies.
it was harder to pick movies than it was to build a query the database however.
